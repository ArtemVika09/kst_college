# Telegram Bot

Телеграм-бот с кнопками для категорий: ПРЕПОДАВАТЕЛЯМ, СТУДЕНТАМ, АБИТУРИЕНТАМ.

## Особенности

- **Кнопка панели** - иконка панели рядом с кнопкой клавиатуры, открывающая веб-приложение с кнопками
- **Постоянная клавиатура** - кнопки внизу экрана для быстрого доступа

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Настройка Web App (кнопка панели)

Для работы кнопки панели нужно разместить файл `panel.html` на веб-хостинге с HTTPS:

### Вариант 1: GitHub Pages (бесплатно)

1. Создайте репозиторий на GitHub
2. Загрузите файл `panel.html` в репозиторий
3. Включите GitHub Pages в настройках репозитория
4. Получите URL вида: `https://yourusername.github.io/repo/panel.html`
5. Замените `WEB_APP_URL` в `kst_bot.py` на ваш URL

### Вариант 2: Netlify/Vercel (бесплатно)

1. Зарегистрируйтесь на [Netlify](https://netlify.com) или [Vercel](https://vercel.com)
2. Загрузите файл `panel.html`
3. Получите URL и замените `WEB_APP_URL` в `kst_bot.py`

### Вариант 3: Локальное тестирование с ngrok

1. Установите [ngrok](https://ngrok.com)
2. Запустите локальный сервер:
```bash
python -m http.server 8000
```
3. В другом терминале запустите ngrok:
```bash
ngrok http 8000
```
4. Скопируйте HTTPS URL из ngrok (например: `https://abc123.ngrok.io`)
5. Замените `WEB_APP_URL` в `kst_bot.py` на `https://abc123.ngrok.io/panel.html`

## Запуск

```bash
python kst_bot.py
```

## Использование

1. Отправьте команду `/start` боту в Telegram
2. Нажмите на иконку панели (сетка 2x2) рядом с кнопкой клавиатуры
3. Выберите категорию из панели или используйте кнопки клавиатуры внизу

## Команды

- `/start` - Показать главное меню с кнопками категорий
