# Решение проблем с подключением

## Ошибка: `telegram.error.TimedOut: Timed out`

Эта ошибка возникает, когда бот не может подключиться к Telegram API.

### Причины и решения:

#### 1. **Нет интернет-соединения**
   - ✅ Проверьте, что у вас есть активное интернет-соединение
   - ✅ Попробуйте открыть https://api.telegram.org в браузере

#### 2. **Telegram API заблокирован**
   - ✅ Используйте VPN
   - ✅ Настройте прокси в коде бота (если необходимо)
   - ✅ Проверьте настройки файрвола

#### 3. **Проблемы с прокси/файрволом**
   - ✅ Отключите файрвол временно для проверки
   - ✅ Проверьте настройки прокси
   - ✅ Убедитесь, что порты не заблокированы

#### 4. **Неправильный токен**
   - ✅ Проверьте, что токен бота правильный
   - ✅ Убедитесь, что бот не был удален в BotFather

### Проверка подключения

Перед запуском бота можно проверить подключение:

```bash
python check_connection.py
```

Этот скрипт проверит:
- Доступность Telegram API
- Правильность токена
- Сетевое соединение

### Автоматические повторы

Бот теперь автоматически пытается переподключиться при ошибках:
- До 5 попыток при таймауте
- Увеличивающаяся задержка между попытками
- Подробное логирование ошибок

### Логи

Все ошибки записываются в логи. Проверьте вывод в консоли для диагностики проблем.

### Дополнительная помощь

Если проблема не решается:
1. Проверьте логи бота
2. Убедитесь, что интернет работает
3. Попробуйте запустить `check_connection.py`
4. Проверьте настройки сети/прокси
